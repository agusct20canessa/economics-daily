
# Economics Daily — Frontend (client)

- Proyecto: Vite + React + Tailwind
- Mobile-first responsive design, ready for AdSense insertion

## Instrucciones rápidas
1. Instalar dependencias: `npm install`
2. Correr dev: `npm run dev`
3. Build: `npm run build`

## Donde insertar AdSense
- En `client/src/components/ArticleCard.jsx` hay un lugar ideal para insertar un bloque `ins` de AdSense debajo del resumen.
- También podes pegar el script principal en `client/index.html` (head) tal como indica Google.

## Optimización móvil
- Diseñado con Tailwind, responsive (grid → single column en móviles).
- Usa fuentes del sistema para rendimiento.
